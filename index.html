📱 Инструкция по установке

Это приложение можно установить на устройство для быстрого доступа как к обычному приложению.

iOS iPhone / iPad

  1. Откройте эту страницу в браузере Safari.
  2. Нажмите на кнопку «Поделиться» (квадрат со стрелкой внизу экрана).
  3. Прокрутите вниз и выберите «На экран «Домой».
  4. Нажмите «Добавить» — иконка появится на рабочем столе.

Android Смартфон / Планшет

  1. Откройте страницу в браузере Chrome.
  2. Нажмите на три точки в правом верхнем углу.
  3. Выберите «Установить приложение» или «На экран "Домой"».
  4. Подтвердите установку — ярлык появится на рабочем столе.

Windows Компьютер / Ноутбук

  1. Откройте страницу в браузере Edge или Chrome.
  2. В адресной строке найдите значок «Установить приложение» (монитор с иконкой).
  3. Нажмите на него и выберите «Установить».
  4. Приложение появится в меню «Пуск» и на рабочем столе (по желанию).

💡 После установки приложение будет работать офлайн (если страница уже загружена).
